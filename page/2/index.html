PHP最强的特性是什么？

什么是懒人函数?

一个功能用JAVA,C#或其他语言有可能要写好多行代码,但是用PHP可能一个函数或一句话就能搞定.

今天学习的第一个懒人函数:

<pre>`int file_put_contents ( string $filename , mixed $data [, int $flags = 0 [, resource $context ]] )

data

返回值:
该函数将返回写入到文件内数据的字节数，失败时返回FALSE
`</pre>

什么是母体?

PHP的一大特性是:脚本语言 不要编译

写完就能运行么?明显不是.

PHP代码要想能够运行,必须有个”母体”.其实就是我们编译好的那个PHP可执行程序

如何让母体运行?

PHP安装目录=/usr/local/sxin/php7
1. /usr/local/sxin/php7/bin/php xxx

2. /usr/local/sxin/php7/bin/php -r “PHP Code”
3. 创建一个文件.第一行写上 #!/usr/local/sxin/php7/bin/php

本课程中 我们采用第三种形式
示例:

  1. sxin

    <?php

    echo 'i am sxin'.PHP_EOL;

    //如果只有PHP代码 是不需要写封口符号的(即:php?>)
    `

    /usr/local/sxin/php7/bin/php sxin
    2. /usr/local/sxin/php7/bin/php -r 'echo "i am sxin".PHP_EOL;'
    3. god

    `#!/usr/local/sxin/php7/bin/php
<?php
    echo 'god'.PHP_EOL;
`

    chmod +x god &amp;&amp;./god

    这个god 会贯穿我们整个教程.为了在任意目录都可以执行这个god.我们把它放到环境变量目录里(/usr/local/bin)

    sudo cp god /usr/local/bin

    实战演练

    PHP如何获取参数.在网页里我们可以通过GET.POST

    但是在命令行里执行要怎么获取参数呢?

    两个全局变量$argc和$argv.

    • $argc是参数的个数(自己也算一个)
    • $argv是参数 它是个数组

    god

    `#!/usr/local/sxin/php7/bin/php
<?php
    $str='i am god';
    echo $str.PHP_EOL;
    echo $argv[$argc-1];
    echo PHP_EOL;
`

本节知识点:

1.速度了解CommonJS

1.CommonJS是一个规范(http://commonjs.org)

譬如:一个单独的文件就是一个模块.使用require来加载模块,返回文件内部的exports对象

大家看是否和NodeJS的写法很像？确实,NodeJS实现并推动了这个规范(一大部分)

有了这个规范JS可以在任何地方运行,不只是浏览器中(未来很多只要兼容CommonJS规范的系统都可以用JavaScript来开发)

课后:请大家回去查询其他规范,如AMD、CMD规范等(譬如看下Require.Js,)

遵循这个规范 然后呢？

这时一个前端构建框架–webpack产生了(模块打包)

它能帮我们把本来需要在服务端运行的JS代码,通过模块的引用和依赖打包成前端可用的静态文件

2.安装

npm install -g webpack

设置配置文件

在项目根目录下新建一个webpack.config.js

module.exports=
{
    entry:[    //entry是入口文件 可以多个
        __dirname+'/sxin.js'
    ],
    output:{
        path:__dirname+'/build/js',
        filename:'build.js'
    }
}
`</pre>

运行webpack

sxin.js

<pre>`var getlib=require('sxin');
console.log(getlib.age);
alert(getlib.age);
`</pre>

安装：curl -fsSL https://get.docker.com/ | sh

注销 重新登陆

查看镜像：`docker search xxx`

下载镜像：`docker pull xxx`

用交互方式创建并进入容器：`docker run -i -t 镜像id /bin/bash`

再次进入容器`docker start 容器ID  docker attach 容器ID`

查看正在运行的容器：`docker ps`

查看所有的容器：`docker ps -a`

杀死所有正在运行的容器
docker kill $(docker ps -a -q)
删除所有容器
docker rm $(docker ps -a -q)

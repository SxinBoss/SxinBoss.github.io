## 需求

如果参数带- 开头,那么说明就是属性.我们直接调用类的静态属性

如果不带- ,那么就说明是方法.我们直接调用静态方法.

常规写法:

<pre>`$p=$argv[1];//先保存一个变量
if(substr($p,0,1)=='-')//判断第一个字符是否是 -
`</pre>

godinit

<pre>`&lt;?php
    class godinit
    {
        static $v="god version is 1.2";
        static $prj_name="";//项目名称
        static $prj_author="sxin";//项目作者
        static function init()
        {
            echo "input your project name?".PHP_EOL;
            self::$prj_name=trim(fgets(STDIN));
            echo "input your project author?".PHP_EOL;
            self::$prj_author=trim(fgets(STDIN));

            return initConfig(self::$prj_name,self::$prj_author);
        }

        static function make()
        {
            $pchar=new Phar("god.phar");
            $pchar-&gt;buildFromDirectory(dirname(__FILE_));
            $pchar-&gt;setStub($pchar-&gt;createDefaultStub('god'));
            $pchar-&gt;compressFiles(Phar::GZ);
        }
    }
`</pre>

这里一定要看清楚:这厮检测的是变量哦

<pre>`&lt;?php

##### 这样的话变量是OK了,那方法咋办呢？

我们又要学习到一个PHP魔法函数 __callStatic($m,$args)

这个方法如果你要用,必须写到类里面并且必须是static

作用:

<pre>`如果你调用了一个未定义的静态方法,则会自动触发这个函数
第一个参数是方法名,第二个参数是方法参数,我们可以用它来做一个容错处理
`</pre>

godinit

<pre>`&lt;?php
    class godinit
    {
        static $v="god version is 1.2";
        static $prj_name="";//项目名称
        static $prj_author="sxin";//项目作者
        static function init()
        {
            echo "input your project name?".PHP_EOL;
            self::$prj_name=trim(fgets(STDIN));
            echo "input your project author?".PHP_EOL;
            self::$prj_author=trim(fgets(STDIN));

            return initConfig(self::$prj_name,self::$prj_author);
        }

        static function make()
        {
            $pchar=new Phar("god.phar");
            $pchar-&gt;buildFromDirectory(dirname(__FILE_));
            $pchar-&gt;setStub($pchar-&gt;createDefaultStub('god'));
            $pchar-&gt;compressFiles(Phar::GZ);
        }

        static function  __callStatic($m,$argv)
        {
            echo 'error function:'.$m;
        }
    }
`</pre>

这样我们在随便执行一下:

`./god sdfji`

它会报:

`error function:sdfji`

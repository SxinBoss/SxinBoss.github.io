PHP魔鬼训练班：第二回合：生猴子

Veröffentlicht am 2016-08-31  |  2016-11-14   |   in PHP . PHP魔鬼训练班

需求:

上节课我们做了一个需求是:如果第二个参数是-v 则显示当前版本

这节课我们加一个需求,如果第二个参数是init,则在当前目录生成一个god.json文件.
文件内容:一对大括号
具体步骤:
1.请在你喜欢的目录下常见一个名字为mypro的文件夹,然后cd进入
2.在mypro文件夹下执行god init
3.一切搞定生成后显示两句话(注意需要换行,要有逼格)
    1、xx of bytes written
    2、god config is created
`</pre>

PHP最强的特性是什么？
- 懒人函数

什么是懒人函数?

一个功能用JAVA,C#或其他语言有可能要写好多行代码,但是用PHP可能一个函数或一句话就能搞定.

今天学习的第一个懒人函数:

###### [file_put_contents](http://php.net/manual/zh/function.file-put-contents.php)

<pre>`int file_put_contents ( string $filename , mixed $data [, int $flags = 0 [, resource $context ]] )
参数:
filename
要被写入数据的文件名。

data
要写入的数据。类型可以是 string，array 或者是 stream 资源（如上面所说的那样）。

返回值:
该函数将返回写入到文件内数据的字节数，失败时返回FALSE
`</pre>

###### [getcwd](http://php.net/manual/zh/function.getcwd.php)

<pre>`返回当前工作目录,否则返回false
`</pre>

在PHP里定义函数:

<pre>`function 函数名(参数1,参数2,参数n)
{
    xxoo;
    如果要return就return.没有也无所谓
}
`</pre>

## 实战演练

至此,我们可以简单的来实现下我们的需求了

<pre>`#!/usr/local/sxin/php7/bin/php
&lt;?php
    $result='';
    if($argc&gt;=2)
    {
        '-v'==$argv[1]&amp;&amp;$result='god version is 1.0';
        'init'==$argv[1]&amp;&amp;$result=fiel_put_contents().' of bytes is written'.PHP_EOL.
        'god config is created';
        echo $result.PHP_EOL;
    }
`</pre>

虽然实现了,不过代码有点丑.我们来优化下

<pre>`#!/usr/local/sxin/php7/bin/php
&lt;?php
    $result='';
    if($argc&gt;=2)
    {
        '-v'==$argv[1]&amp;&amp;$result='god version is 1.0';
        'init'==$argv[1]&amp;&amp;$result=genConfig();
        echo $result.PHP_EOL;
    }

    function genConfig()
    {
        return fiel_put_contents().' of bytes is written'.PHP_EOL.'god config is created';
    }
`</pre>

嗯...现在看起来比之前好多了,不过问题又来了,当我们god功能越来越多,那么要写的函数就越来越多,所有函数都放在god里.看着就头疼,于是我们要写一个专门的函数库.

##### god_func

<pre>`&lt;?php
    function genConfig()
    {
        return fiel_put_contents().' of bytes is written'.PHP_EOL.'god config is created';
    }
`</pre>

##### god

<pre>`#!/usr/local/sxin/php7/bin/php
&lt;?php
    require('god_func');
    $result='';
    if($argc&gt;=2)
    {
        '-v'==$argv[1]&amp;&amp;$result='god version is 1.0';
        'init'==$argv[1]&amp;&amp;$result=genConfig();
        echo $result.PHP_EOL;
    }
`</pre>

既然我们是用的PHP7
那么我们就要适当学习PHP7的特性
- PHP7允许在函数中增加返回值.譬如string,int,array,object等(这些类型是PHP7才有的,我们暂时先简单了解即可)

<pre>`function 函数名():string //注意冒号
{
}
`</pre>

那么好了,我们把刚才的god_func里的函数改下

<pre>`&lt;?php
    function genConfig():string
    {
        return fiel_put_contents().' of bytes is written'.PHP_EOL.'god config is created';
    }
`</pre>

但是问题又来的,这样改了之后,PHP5又运行不了了.
所以我们还要做一个兼容函数库.把刚才的god_func改成god_func7.然后在新建一个goc_func5

###### god_func7

<pre>`&lt;?php
    function genConfig():string
    {
        return fiel_put_contents().' of bytes is written'.PHP_EOL.'god config is created';
    }
`</pre>

###### god_func5

<pre>`&lt;?php
    function genConfig()
    {
        return fiel_put_contents().' of bytes is written'.PHP_EOL.'god config is created';
    }

然后在我们的god程序中判断当前的PHP版本加载不同的函数库.(PHP版本常量: PHP_VERSION)

  • 如果是PHP7则加载(require)god_func7文件,否则rquire god_func5

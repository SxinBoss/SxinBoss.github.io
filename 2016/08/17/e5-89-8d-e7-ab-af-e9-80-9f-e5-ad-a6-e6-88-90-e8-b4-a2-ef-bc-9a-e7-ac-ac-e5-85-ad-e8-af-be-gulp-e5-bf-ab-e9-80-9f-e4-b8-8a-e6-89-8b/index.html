前端速学成财：第六课-gulp快速上手

Veröffentlicht am 2016-08-17  |  2016-11-14   |   in 前端专区 . 前端速学成财

本节知识点:

1.快速理解gulp

2.安装和测试

3.执行我们第一个任务

通过便面的”开脑”课时,我们会发现:

在前段开发过程中有很多事是我们”纯手工”很难或者很麻烦才能搞定的,因此需要借助一些优秀的第三方工具.

1.譬如JS文件的”编译”(这里还包含了不同语法的兼容)

2.JS和CSS的合并

3.文件的压缩

4.多人协作开发

5.即时运行和监听等

gulp只是一个主框架,他还要和许多插件配合使用.

生成package.json:npm init

npm install -g
如果不加-g 则会安装到当前目录下
加上-g就会安装到一个全局目录里
我们可以通过npm config get prefix 查看全局目录
也可以通过npm config set prefix "目录路径" 来设置
一般来说,工具性模块:我们需要在任意目录都想用的工具,使用全局安装.
`</pre>

#### **安装gulp**

<pre>`npm install -g gulp
npm install gulp --save-dev
`</pre>

#### **gulp配置文件**

要正确运行首先要建一个配置文件.

在当前项目目录下创建gulpfile.js(这个所谓的配置文件是可以写代码的,不是想文本配置文件那样)

<pre>`var gp=require("gulp");
gp.task("mytask",function(){
    console.log("my first gulp task");
});

执行gulp任务

gulp mytask
